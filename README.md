<h1 align="center">
  ITU-minitwit
</h1>
<p align="center">
ITU Minitwit application written in Go with the Gin library.
<br/>

## ⚡️ Quickstart!

```sh
cd minitwit-go
docker-compose up
```
`docker-compose up` runs both the database and the server


If files have been changed, then instead run
```sh
docker-compose up --build
```
